1. Cenário
Título

Registrar chamado de suporte

Objetivo

Permitir que o usuário registre uma solicitação de suporte.

Atores

Usuário

Recursos

Sistema Help Desk

Episódios

Usuário acessa o sistema

Usuário realiza login

Usuário seleciona a opção "Novo Chamado"

Usuário preenche os dados do chamado

Sistema registra o chamado com status "Aberto"
